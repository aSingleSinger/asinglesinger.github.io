1、ajax简介

  1. ajax全称为Asynchronous JavaScript And XML，就是异步的JS 和XML
  2. 通过AJAX可以再浏览器中向服务器发送异步请求，最大的优势：无刷新获取数据
  3. ajax不是新的编程语言，而是一种将现有的标准组合在一起使用的新方式

2、XML简介

  1. 可扩展标记语言

  2. 被设计用来传输和存储数据

  3. XML和HTML类似，不同的是HTML中都是预定义标签，而XML中没有预定义标签，全都是自定义标签，由来表示一些数据。就是自己给标签取名字

  4. 现在被json取代了

3、ajax特点

1、优点

  • 可以无需刷新页面而与服务器端进行通信
  • 允许根据用户事件来更新部分页面内容

2、缺点

  • 没有浏览历史，不能回退
  • 存在跨域问题（同源）不能从一个网站访问另一个网站
  • SEO不友好(不能爬取数据)

5、express框架使用

  1. 在所在目录终端下输入
npm init --yes
  1. 安装express
npm i express
  1. 编写js文件

app.get app.post app.all代表接收所有形式的请求

// 1. 引入express
const express = require('express');

// 2. 创建应用对象
const app = express();

// 3. 创建路由规则
// request是对请求报文的封装,reponse是对相应报文的封装
app.get('/',(request,response)=>{
  response.send('HELLO EXPRESS');
});

// 4. 监听端口启动服务
app.listen(8000,()=>{
  console.log("服务器已经启动,8080端口监听中...");
})

6、通过原生js进行ajax操作

页面：不刷新页面的情况下修改页面

<html>
  <head>
    <meta charset="utf-8">
    <title>AJAX GET请求</title>
    <style>
      #result{
        width:200px;
        height:100px;
        border:solid 1px #90b;
      }
    </style>
  </head>
  <body>
    <button>点击发送请求</button>
    <div id="result"></div>
  </body>
  <script>
    const btn = document.getElementsByTagName('button')[0];
    const result = document.getElementById('result');
    // 绑定事件
    btn.onclick = function(){
      // 1. 创建对象
      const xhr = new XMLHttpRequest();
      // 2. 初始化 设置请求方式和 url
      xhr.open('GET','http://127.0.0.1:8000/server');
      // 3. 发送
      xhr.send();
      // 4. 事件绑定 处理服务器端返回的结果
      // readystate 是 xhr对象中的属性,表示状态
      // 0(未初始化) 1(open方法调用完毕) 2(send方法调用完毕) 3(服务端返回部分结果) 4(服务端返回所有结果)
      xhr.onreadystatechange = function(){
        // 判断 服务端返回了所有的结果
        if(xhr.readyState === 4){
          // 判断响应状态码
          // 2xx 都表示成功
          if(xhr.status >= 200 && xhr.status <= 300){
            // 处理结果 行 头 空行 体
            // 1. 响应行
            // console.log(xhr.status);
            // console.log(xhr.statusText);
            // console.log(xhr.getAllResponseHeaders());
            // console.log(xhr.response);// 响应体
            result.innerHTML = xhr.response;
          }
        }
      }
    }
  </script>
</html>
// 1. 引入express
const express = require('express');

// 2. 创建应用对象
const app = express();

// 3. 创建路由规则
// request是对请求报文的封装,reponse是对相应报文的封装
app.get('/server',(request,response)=>{
  //设置响应头 设置允许跨域
  response.setHeader('Access-Control-Allow-Origin','*');
  
  // 设置响应体
  response.send('HELLO EXPRESS');
});

// 4. 监听端口启动服务
app.listen(8000,()=>{
  console.log("服务器已经启动,8080端口监听中...");
})
  1. 设置get参数
xhr.open('GET','http://127.0.0.1:8000/server?a=100&b=200');
  1. ajax发送post请求
xhr.open('POST','http://127.0.0.1:8000/server');
// 格式灵活,服务器端能处理即可 传输信息
xhr.send('a=100&b=200');
  1. 设置请求头，身份校验的信息通常放在这里
// 设置请求头
// Content-Type 设置请求体内容类型
xhr.setRequestHeader('Content-Type','application/x-www-form-urlencoded');

xhr.setRequestHeader('name','2dog');
在服务器端如果想要接收自定义的内容 比如name,需要设置response的响应头(自定义的setHeader也可以传入)
response.setHeader('Access-Control-Allow-Headers','*');
  1. 服务器端响应JSON数据

服务器端

const data = {
    name:'dog'
};
let str = JSON.stringify(data);

response.send(str);

前端接收

1. 手动转化
let data = JSON.parse(xhr.response);
result.innerHTML = data.name;
2. 改变响应体类型
xhr.responseType = 'json';
xhr.onreadystatechange = function(){
    if(xhr.readyState == 4){
        if(xhr.status >= 200 && xhr.status <= 300){
           // 直接使用
            result.innerHTML = xhr.response.name;
        }
    }
}

8、ajax解决IE缓存问题

ajax在服务端更新后，理应浏览器的数据也被更新，但是IE浏览器存在缓存问题(chrome就没有),所以需要处理，如下是一种方式

xhr.open('GET','http://127.0.0.1:8000/ie?t='+Date.now());

10、手动取消请求

请求成功需要时间,如果在请求成功前想要关闭，可以使用该方法

let x = null;
btns[0].onclick = function(){
    x = new XMLHttpRequest();
    x.open('get','http://127.0.0.1:8000/delay');
    x.send();
}
// 手动关闭 调用abort()方法
btns[1].onclick = function(){
    x.abort();
}

11、求情重复发送问题

const btns = document.querySelectorAll('button');
let x = null;
// 标识变量 是否正在发送AJAX请求
let isSending = false;
btns[0].onclick = function(){
    // 如果正在发送取消原来的
    if(isSending){
        x.abort();	
    }
    x = new XMLHttpRequest();
    isSending = true;
    x.open('get','http://127.0.0.1:8000/delay');
    x.send();
    x.onreadystatechange = function(){
        // 只需判断状态码是否为4即可
        if(x.readyState === 4){
            isSending = false;
        }
    }
}
